# sbin

这里放置安装需要的二进制文件，请移步百度网盘下载`k8s-1.9.tar.gz`二进制文件压缩包解压放置到此目录下：

https://pan.baidu.com/s/1hsZViQw

按如下结构组织文件:

- bridge
- calicoctl
- cfssl
- cfssl-certinfo
- cfssljson
- conntrack.deb
- docker-ce.deb
- docker-compose
- etcd
- etcdctl
- flanneld
- host-local
- kube-apiserver
- kube-controller-manager
- kube-proxy
- kube-scheduler
- kubectl
- kubelet
- libltdl7.deb
- loopback
- mk-docker-opts.sh

版本：

- cfssl: 1.2
- docker-ce: 17.12.0
- docker-compose: 1.18
- etcd: 3.2.13
- kubernetes: 1.9.2
- flannel: 0.9.1
